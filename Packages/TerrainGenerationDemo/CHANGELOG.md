# Changelog

All notable changes to TerraForge are documented in this file.

## [1.0.0] - 2026-04-15

- Added deterministic procedural terrain heightfield generation with mesh output.
- Added terrain zone classification for water, beach, grassland, rock, and mountain regions.
- Added smoothed vertex-color zone visualization and preview texture generation.
- Added deterministic placeholder prop placement with package-local prefabs.
- Added editor tooling for preset editing, previewing, generating, and exporting terrain.
- Added mesh, preview PNG, prefab, and TerrainData export support.
- Added the TerrainGenerationDemo scene as a package-local example scene.
